# AI Talent Hub for Students of Bangladesh

A national-scale, AI-first talent identification, development, and opportunity-matching platform designed to prepare Bangladeshi students for the global AI economy.

This repository follows a **spec-driven development** approach using Cursor + Claude Code.

## Objectives
- Identify student potential early
- Build AI-age skills
- Create verified AI Talent Passports
- Connect talent to global opportunities
- Enable Bangladesh as a global AI talent exporter
